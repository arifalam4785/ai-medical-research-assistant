# 🩺 CuraLink

**CuraLink** — A full-stack MERN application.

## 📁 Project Structure

```
curalink/
├── frontend/    → React (Vite)
├── backend/     → Node.js + Express
└── README.md
```

## 🚀 Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

Server runs at `http://localhost:5000`

### Frontend

```bash
cd frontend
npm install
npm run dev
```

App runs at `http://localhost:5173`

| Layer    | Technology       |
| -------- | ---------------- |
| Frontend | React + Vite     |
